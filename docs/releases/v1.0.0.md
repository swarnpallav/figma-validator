# v1.0.0 — Initial Release

## Features

- Figma plugin to extract design styles
- Chrome extension to inspect DOM styles
- Shift + Click element validation
- Comparison for:
  - spacing
  - typography
  - colors
  - border radius
- Visual overlay for validation results

## Architecture

- Figma Plugin API
- Chrome Extension (Manifest v3)
- postMessage communication bridge
- chrome.storage for snapshot transfer

## Status

- Figma plugin approved and published
- Chrome extension published (unlisted)
