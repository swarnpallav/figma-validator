## 🚀 v1.2.0 — Intelligent UI Validation

Figma Validator now goes beyond detecting UI mismatches — it helps you understand *why* they happen and how to fix them.

---

### ✨ What's new

- 🧠 Hybrid validation (geometry + CSS insights)
- 💡 Likely issue detection for mismatches
- 🔧 Actionable suggestions to fix UI inconsistencies
- 📏 Improved geometry comparison (width & height with diff)

---

### 🧠 Improvements

- Reduced noise by showing CSS details only for mismatched elements
- Clearer mismatch visualization with inline diff values (+px / -px)
- Better accuracy in detecting layout inconsistencies

---

### 🐛 Fixes

- Improved handling of nested elements
- Fixed edge cases in spacing and size calculations

---

### 🎯 Why this matters

Instead of manually inspecting styles and guessing what went wrong, you can now:

→ Detect mismatches instantly  
→ Understand the root cause  
→ Fix issues faster  

---

### ⚠️ Note

This release builds upon earlier geometry-based validation improvements introduced in v1.1.0.
